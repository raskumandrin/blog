The official card reading application only works on Windows.

http://ca.mup.gov.rs/ca/ca_cyr/start/kes/

How to read the card on a Mac?

1. Install VirtualBox. Everything worked for me only after launching VirtualBox as root.

2. Install Windows 11 inside it. The ISO image can (and should!) be downloaded from Microsoft's website.

3. In Windows, install all the applications from the MUP's list

4. Forward the USB port

5. Inside the running VirtualBox, with the card reader connected, you need to connect the USB device

After that, you can read the card and print the readout.
