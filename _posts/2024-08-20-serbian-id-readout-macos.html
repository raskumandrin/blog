Официальное приложение чтения с карты работает только под Windows.

http://ca.mup.gov.rs/ca/ca_cyr/start/kes/

Как прочитать карту с Мака?

1. Установить VirtualBox.У меня всё заработало только после запуска Virtual Box из под root.

2. Установить внутрь Windows 11. ISO образ можно (и нужно!) скачать с сайта Microsoft.

3. В Windows установить все приложения по списку от МУПа

4. Пробросить USB порт

5. Внутри запущенного VirtualBox при подключенном карт-ридере нужно подключить USB устройство

После этого можно читать карту и делать печать readout.
