Официальное приложение ePorezi только под Windows.

https://www.purs.gov.rs/sr/e-porezi/aplikacija-eporezi.html

Как использовать ePorezi на MacOS?

1. Установить VirtualBox.У меня всё заработало только после запуска Virtual Box из под root.

2. Установить внутрь Windows 11. ISO образ можно (и нужно!) скачать с сайта Microsoft.

3. В Windows установить приложение ePorezi.

4. Пробросить USB порт

5. Внутри запущенного VirtualBox при подключенном USB-ключе нужно подключить USB устройство

После этого можно запустить ePorezi и пройти аунтефикацию по ключу.

При обращении в компанию для подготовки налоговой декларации «под ключ» лучше вместо передачи USB-ключа предоставить вот тут сотруднику доступ:
